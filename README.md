# AI Agent Playwright Project

This is a Playwright testing project.

## Installation

Run `npm install` to install dependencies.

Run `npx playwright install` to install browsers.

## Running Tests

Run `npm test` to run tests in headless mode.

Run `npm run test:headed` to run tests in headed mode.

Run `npm run test:ui` to run tests with UI.

## Reports

After running tests, open `playwright-report/index.html` to view the report.
